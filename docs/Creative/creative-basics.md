# Creative Basics
In this section you will learn how the RackMC Creative server works!

The RackMC Creative server is a relaxed, building-focused experience designed to give you complete freedom to create without limitations. With access to large personal plots and an extensive set of building tools, players can design anything from small projects to massive, detailed builds.

Whether you are experimenting with new ideas or refining complex builds, the Creative server provides a space to build, share, and bring your imagination to life.
