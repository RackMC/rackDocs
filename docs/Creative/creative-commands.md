## Creative

`/plot help`
Description: Get the help menu of PlotSquared.

`/plot info`
Description: Display the plot info.

`/plot list <forsale | mine | shared | world | top | all | unowned | unknown | player | world | done | expired | area>`
Description: Display a list of the existing plots on the server.

`/plot target <<X;Z> | nearest>`
Description: Target a plot with your compass.

`/plot [[world;]X;Z] caps`
Description: Show plot mob and entity caps.

`/plot home`
Description: Teleport to your own plot(s).

`/plot visit <player>`
Description: Use this command to go to the plot of a specific player.

`/plot middle`
Description: Teleport you to the center of the plot.

`/plots comment`
Description: Comment on a plot.

`/plots inbox`
Description: List the comments for a plot / delete a message or clear the comment list.

`/plot claim`
Description: Claim the current plot.

`/plot auto`
Description: Claim the nearest plot in your world.

`/plot setowner <player>`
Description: Set the plot owner.

`/plot add <player | *>`
Description: Add one or more players to the plot.

`/plot trust <player | *>`
Description: Add one or more players to the plot.

`/plot remove <player | *>`
Description: Remove a player from a plot.

`/plot deny <player | *>`
Description: Deny a user from entering a plot.

`/plot kick <player | *>`
Description: Kick a player from your plot.

`/plot merge <all | n | e | s | w> [removeroads]`
Description: Merge the plot with another plot(s).

`/plot unlink [createroads]`
Description: Unlink a merged plot.

`/plot set home`
Description: Set the plot home.

`/plot alias set`
Description: Set the "plot name".

`/plot alias remove`
Description: Remove the "plot name".

`/plot set description`
Description: Set the plot description.

`/plot music`
Description: Play music in a plot.

`/plot set biome`
Description: List all possible biomes or change the plot biome.

`/plot set flag <arg>`
Description: Manage the flags of a plot.

`/plot done`
Description: Mark a plot as "done". This means that even the plot owner and plot members can no longer build on the plot.

`/plot continue`
Description: This removes a previously set "done" marker so that the players can continue building on the plot.

`/plot toggle <chat | clear-confirmation | time | titles | worldedit>`
Description: Toggle per user settings.

`/plot set <biome | alias | home | floor | wall | all | air | main | middle | outline | border> <value...>`
Description: Set a plot value.

`/plot copy`
Description: Copy a plot.

`/plot move`
Description: Move a plot.

`/plot swap`
Description: Swap two plots.

`/plot backup <save | list | load>`
Description: Manage plot backups.

`/plot clear`
Description: Clear the plot you stand on. It doesn't reset any plot settings or flag (except the biome setting).

`/plot delete`
Description: Delete the plot you stand on.

`/plot load`
Description: List the plot snapshots (after: `/plot save`) or load a snapshot.

`/plot download`
Description: Download your plot.

`/plot schematic <save | saveall | paste>`
Description: Use and manage the plot schematics.

`/plot like`
Description: Like a plot.

`/plot like`
Description: Dislike a plot.

`/plot rate`
Description: Rate the plot.
