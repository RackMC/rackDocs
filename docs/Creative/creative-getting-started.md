# Getting Started

When joining the server for the first time, your main goal is to claim a plot and begin building. The Creative server removes survival limitations, allowing you to focus entirely on designing and creating.

## Claim Your Plot

To get started, find an open plot and claim it:

`/plot claim`

If you do not have a specific location in mind, you can automatically claim the next available plot:

`/plot auto`

Once claimed, the plot is yours to build on and customize.

## Navigating Plots

You can quickly return to your plot at any time:

`/plot home`

To visit another player’s build:

`/plot visit <player>`

To view information about your current plot:

`/plot info`

## Building and Managing Your Plot

You have full control over your plot and can modify it to suit your needs. Some useful commands:

Add other players to help build:

`/plot add <player>`

Trust players for broader access:

`/plot trust <player>`

Remove or restrict players:

`/plot remove <player>`

`/plot deny <player>`

Set your plot home location:

`/plot set home`

Change your plot’s name or description:

`/plot alias set`

`/plot set description`

Customize terrain and appearance:

`/plot set biome`
`/plot set <value>`

## Expanding Your Build

If you need more space, you can merge adjacent plots:

`/plot merge`

This allows you to create larger builds without interruption.

## Finishing and Showcasing

When your build is complete, you can mark it as finished:

`/plot done`

This prevents further edits and allows others to view your work as intended. You can also share your plot with others or explore builds across the server using:

`/plot list`

The Creative server is designed to make building simple and accessible while giving you the tools to create complex and detailed projects.
