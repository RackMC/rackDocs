After being down for a couple hours last night, it seems the the server is back up and running with no issues. Here is a list of feature changes and bug fixes since the last post. If you find any more issues, make sure to open a ticket in our [discord](https://discord.rackmc.net). 

## Major Changes
### Lands
- We have started work on a house at spawn that will give basic tutorials for many aspects of the server
- The number of homes a player can set has been increased from 3 to 5
- Voting now has a reward of Vote Keys that can be used at the Vote Crate at `/spawn`
    - NOTE: This is a temporary reward and will be improved upon later

## Known Issues
### Lands
- The Server Bank does not fill up automatically
- `/rtp` does not work (use `/wild` for now)
- The MCMMO Mining skill levels up too quickly
### Creative
- You can not currently spawn any mobs in your plots

## Bug Fixes
### Network
- Players now show up as AFK in the tab list, as well as in chat
- Improved the maintenance MOTD in the server list
- Monsters can no longer spawn in the spawn land
- `/discord` now provides a link to the discord server
### Creative
- Achievements are now disabled
