It is now week 9 of the RackMC Open Beta and week 1 of the Factions Season 0 playtest! We are back to weekly updates while Season 0 is running to make sure up to date info is given on the state of the server. Factions definitely has its share of bugs, so if you encounter any problems while playing on the server, make sure to let us know so we can get to work on it!

Here is a list of what has happened over the past week, as well as any planned changes for this coming week:

## Upcoming Changes
### Factions
- **Factions Season 0 will be ending June 8th @ 6pm EST!**
- We will be reworking all of the shop prices so they are more balanced and make progression more fun and fair

## Bug Fixes
### Network
- The server works again!
- Our server stats page seems to be working (`/plan register` to make an account and view it)
### Factions
- Silk touch now works on spawners
- Chat now shows player's faction in hover text
- /pay now works
- Diamond blocks now add more value to factions top list

## Known Issues
### Network
- Tags are currently equipped per server instead of globally
- The playercounts on the scoreboard don't always update correctly
### Lobby
- Hunger does not drain during duels
### Lands
- `/rtp` does not work (use `/wild` for now)
- The MCMMO Mining skill levels up too quickly (This will be fixed on the full release)
### Factions
- Some versions have issues connecting (Should be mostly fixed)
- /f home does not work
- /baltop does not work
