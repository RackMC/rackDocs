Week 6 of the RackMC Open Beta is here (one day late). The server has been down for about a week, so there are no changes, but I thought I'd take this opportunity to tell you where we are at.

## Current State of RackMC
The current server host is having some issues. We are in the process of finding a new host. Once we do, it will take some time to transfer all the files (over 100gb!) and make sure everything is working once again. We don't have an ETA right now, but we will let you all know as soon as we make a decision and start this process.

## Factions Season 0
Once we have the server back up, we will want to take about a week to make sure it doesn't have the same issues as before. Last thing we want is to launch an unstable Factions playtest. If everything seems to be in working order, we will set a date and get Season 0 going!

## Future Update Posts
In terms of these update posts, we were starting to get to a point where there weren't a lot of changes to add to the list. The server was in a really good place before everything went wrong. After this post, we will start making monthly posts on the 1st of each month (starting May 1st). If we have any special announcements, changes, or other important info, we will make a separate post for that.

## Going Forward
This is a open beta after all, so things like this are expected, but they are still quite annoying. We wish the server didn't have to be down so long, but we haven't had much of a choice unfortunately. We are working on a backend infrastructure that should allow the server have minimal downtime if we did have a problem like this again. This is also a work in progress, but we hope it will be implemented before the official launch of RackMC.

Either way, we hope this will all be sorted out soon and hope to see you for the Factions playtest once it launchs!
