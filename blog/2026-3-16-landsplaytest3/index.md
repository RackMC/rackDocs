Today begins our 3rd Lands Playtest. This one will be different however, as it will be up until we officially launch the server. The playtest will begin at 8pm EST and run until the official launch of the server.

Due to how long its been since our last playtest and the fact that we haven't worked on the server much since then, we have essentially had to rebuild Lands from the ground up. This means that there are a some new features, some removed features, and a lot of broken ones. I just wanted to make this post so anyone who is playing will have a quick reference of we the Lands server, and the network as a whole can offer.

*NOTE: We will try to keep this list updated with all broken or missing features, so if you notice that something is not working quite right and it is not listed here, make sure to report it in our [discord](https://discord.rackmc.net).*

## Lands
### New Features
#### Raw Copper Economy
- Raw copper can be mined and sold at spawn to be used as a digital currency
### Broken Features
#### Self Sustaining Economy
- Money spent on land purchases and upkeep does not currently go into a server account
- Jobs do pay out from server account, but money is add manually at the moment
### Future Features
#### Lands Spawn
- For the full release (or sooner) we plan to add a spawn area with helpful NPCs, items, or tutorials
#### ATM/Banker
- We plan to have a banker or ATM at spawn to facilitate easy raw copper to digital copper conversions
#### Proximity Voice Chat
- We (potentially) plan to add a proximity voice chat to the lands server
- This may be added during the beta for testing, but may or may not make it into the full release
#### Discord Bridge
- We may add a minecraft to discord chat bridge for players who are not online to chat through discord
#### Bluemap
- An interactive map of the entire lands server
- This is one that is almost ready, but not quite there
