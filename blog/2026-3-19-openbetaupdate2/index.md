The RackMC Beta has now been going on for a few days, so I wanted to give you all yet another update of whats been going on. If you find any more issues, make sure to open a ticket in our [discord](https://discord.rackmc.net). 

## Major Changes
### Lands
We have implemented server voting:
- We have added the /vote command. Currently this will just give you a list of server list sites you can vote for our server on, but eventually it will be set up with some pretty good rewards!
- For the time being, if you let us know that you voted on all the sites on the [RackMC Discord](https://discord.rackmc.net), we will give you **1000 Raw Copper** ($) on the Lands server. You are able to do this every 24 hours.
- Voting helps us grow the server, so any voting that you do is greatly appreciated. 

## Known Issues
### Lands
- You do not recieve vote points for voting

## Bug Fixes
### Lands
- Clear lag has been extended to 15 mins and has a proper countdown
- You can now use an elytra in the spawn land
### Creative
- World edit now works in your owned plots
- You can now set flags (such as weather clear or time day) in your plots
- The tablist now shows the server player count properly
