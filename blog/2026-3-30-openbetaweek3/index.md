It is now week 3 of the RackMC Open Beta. We have made one big change to the Lands server and have some news about the Factions server, so read below!

Here is a list of all the bug fixes and changes that have been made since last week. If you find any more issues, make sure to open a ticket in our [discord](https://discord.rackmc.net). 

## Major Changes
### Network
- Added Factions to the server count scoreboard (you can turn this off with /sb)
### Lobby
- Added daily, weekly, and monthly mystery boxes with unlockable cosmetics
- Setup duels in the arena (wip). Join by left clicking the sign
### Lands
- We have removed the server bank account for jobs. Now jobs will have an unlimited payout (for now)
- A shopping district has been created for players to have a localized shopping area
- Added [vanilla tweaks](https://vanillatweaks.net/picker/datapacks/) datapacks
    - silence mobs
    - age lock
    - player head drops
    - mob head drops
    - fast leaf decay
    - double shulker shell drops
    - cauldron concrete and mud
    - bat membranes (disabled phantoms)
- New players now start with a balance of $300 copper
### Factions
- Started work on the Factions server with a potential playtest on the horizon :eyes:

## Known Issues
### Lands
- `/rtp` does not work (use `/wild` for now)
- The MCMMO Mining skill levels up too quickly
- The smelting part of the crafter job does not pay out for an auto smelter
- The fortune enchanment on a pickaxe conflicts with MCMMO double drops

## Bug Fixes
### Network
- You can now join all servers on any version between 1.8 and 26.1 (though I'd recommend 1.21.11 for now)
### Lands
- PVP is now off by default
- Elytra and ender pearl use is now enabled at spawn and the shopping district
- Added a 3 sec wait for /spawn, /home, and /warp to bring it line with the other tp commands (im sorry)
