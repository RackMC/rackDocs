Week 4 of the RackMC Open Beta is here. We are also happy to announce that we will be starting our Factions Playtest **this Saturday at 6pm EST**!! It will run from April 11th - May 9th, lasting 30 days total. We have additional information about it [here](https://rackmc.net/blog/factionsplaytest) if you would like to learn more!

We have also launched our Netherite Rank, which is our limited time, permanent donator rank that will give all current and future perks on the server! You can read more about that [here](https://rackmc.net/blog/donatorranks)

Here is a list of all the bug fixes and changes that have been made since last week. If you find any more issues, make sure to open a ticket in our [discord](https://discord.rackmc.net). 

## Major Changes
### Network
- We are currently working on a completely custom chat plugin. This will allow for a lot of new features, but will likely have some bugs while we continue to work on it. This is currently only available on lobby, lands, and creative
### Lobby
- Updated the daily rewards
- Added vote rewards for the lobby (`/vote`)
### Lands
- Added an enderchest at spawn
- You can now claim land in the nether and the end (the main end island is not claimable)
### Factions
- Playtest starting April 11th at 6pm EST
### Website
- We have updated the Lands and Factions [documentation](https://rackmc.net/docs/) (still a work in progress)
- Added Ranks to the [store](https://store.rackmc.net/) (only Netherite is purchasable at this time)
- Added FREE tag to [store](https://store.rackmc.net/)

## Known Issues
### Lobby
- Using the server selector on version 26.1 will disconnect you from the server (use NPCs for now)
- Right clicking the duels sign on versions prior to 1.20 will disconnect you from the server
### Lands
- `/rtp` does not work (use `/wild` for now)
- The MCMMO Mining skill levels up too quickly
- The smelting part of the crafter job does not pay out for an auto smelter
- The fortune enchanment on a pickaxe conflicts with MCMMO double drops
- A client side issue with the `/sit` command

## Bug Fixes
### Network
- Reformatted auto restart messages
- Fixed the `/discord` command on all servers
