Week 5 of the RackMC Open Beta is here  (one day late). A lot has happened over the past week.

For those who missed it, the Factions Season 0 playtest has been postponed until **Friday, April 17th @ 6pm EST**. It will still last 30 days, ending on May 15th. We hope to have Lands and Creative back up over the next couple of days.

Here is a list of all the bug fixes and changes that have been made since last week. If you find any more issues, make sure to open a ticket in our [discord](https://discord.rackmc.net). 

## Major Changes
### Network
- Donator ranks (Netherite) now have access to 10 new tags and customizable particles in the lobby
### Lands
- Limited versions to 1.20 - 26.1 to prevent major bugs
### Website
- We have updated the Creative Docs

## Known Issues
### Network
- Tags are currently equipped per server. With RackChat, we should be able to make it equipped globally
### Lobby
- Hunger does not drain during duels
- The playercounts on the scoreboard don't always update correctly
### Lands
- `/rtp` does not work (use `/wild` for now)
- The MCMMO Mining skill levels up too quickly
- The smelting part of the crafter job does not pay out for an auto smelter
- The fortune enchanment on a pickaxe conflicts with MCMMO double drops
- A client side issue with the `/sit` command
### Factions
- The playtest isn't live yet :(
### Creative
- The playercounts on the scoreboard don't always update correctly

## Bug Fixes
### Network
- RackChat is on all servers and seems to be working well (let us know if you find any issues)
### Lobby
- Using the server selector on version 26.1 will disconnect you from the server (use NPCs for now)
- Right clicking the duels sign on versions prior to 1.20 will disconnect you from the server
- Players can no longer leave the main spawn area
- Duels is now using 1.8 combat instead of 1.9
### Lands
- Combat log messages are formatted correctly now
